Simplified Water Allocation Models (SWAM)

The Surface Water Program maintains master copies of the Simplified Water Allocation Models. Use this page to download models for the Edisto, Saluda and Salkehatchie River basins.

Public Release | Background | System Requirements | Model Updates | Distribution | Notice and Disclaimer | Download Models | Contact Information

Public Release

The South Carolina Department of Natural Resources (SCDNR) is pleased to announce the public release of the Simplified Water Allocation Models (SWAM) for the Edisto, Saluda and Salkehatchie river basins. These models are being made available via this FTP Site. A folder is listed for each basin and each folder includes the following files:

  1. Excel file, with extension .xlsm containing the SWAM Baseline Model.
  2. South Carolina Surface Water Quantity Model Report for the basin.
  3. Supplemental Technical Memorandums, if any, regarding any updates or changes to the model since the initial model report was released.
  4. An SCDNR Model Review Technical Memorandum, if available, describing any important limitations or other considerations for using the SWAM model in a given basin.
Users should carefully review the modeling report and any technical memorandums included in each basin folder prior to using the model. Users are also strongly encouraged to review the SWAM User’s Manual (v5.0) that is available on the FTP site under the User’s Manual folder.

Background

The SCDNR, in collaboration with the South Carolina Department of Health and Environmental Control (SCDHEC), awarded a contract to CDM Smith, Inc. in August 2014 for the purpose of developing surface water quantity models using the SWAM modeling tool. The SWAM models are intended to support the development of regional water plans for each of the eight major river basins in the State. Information on the State’s water planning efforts can be found on the SCDNR website and at scwatermodels.com.

System Requirements

Minimum system requirements for using the SWAM models include Microsoft Excel 2013 and a computer with 8 GB of RAM.

Model Updates

The SCDNR has made tremendous efforts to ensure the accuracy of data included in the model but cannot guarantee that data inputs are error free. Models will be updated as necessary and a log describing any changes or updates to the model will be included in each basin’s subfolder. To ensure access to the most recent model, users are encouraged to access the model directly from this public site.

Distribution

Each individual user should download his or her model(s) of interest directly through this public access site. The distribution of the models to other potential users is strongly discouraged.

Download Models

You can access the FTP site by filling out the form below. You will be asked for basic information about how you plan to use the models. After completing this survey, you will be given the URL to the FTP Site.

Once on the FTP site, click on the name of the folder corresponding to the basin of interest. Select the file(s) of interest and then click, hold and drag the file to the desired folder on the local computer.
